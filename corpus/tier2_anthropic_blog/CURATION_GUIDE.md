# Tier 2 Content Curation - Anthropic Blog/News

For Tier 2, we're curating official Anthropic content that isn't core documentation:
- Blog posts about Claude Cowork
- Release announcements
- Case studies and examples

## Sources to Check

**Anthropic Blog/News:**
- https://www.anthropic.com/news
- https://www.anthropic.com/research
- Search for "Cowork" announcements

**What to look for:**
- Cowork launch announcements
- Feature updates (computer use, plugins, etc.)
- Use cases and examples
- Best practices from Anthropic

```json
}
```
